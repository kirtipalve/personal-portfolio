# Kirti Palve - Personal Portfolio

Personal portfolio website showcasing my projects, experiences, and skills.

Built with Next.js 15, TypeScript, and TailwindCSS. Features Server Side Rendering for optimal performance.

## About

This portfolio showcases my work as a Software Engineer, including my experience at Adobe, leadership at TEDx, and studies at Carnegie Mellon University.

## Tech Stack

- **Framework:** Next.js 15 with Turbopack
- **Language:** TypeScript
- **Styling:** TailwindCSS v4
- **Deployment:** Vercel
- **Icons:** Lucide React

## Running Locally

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```
